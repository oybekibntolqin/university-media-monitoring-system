# University Media Monitoring System

## 📌 Overview

A backend system for managing university media publications.

This platform allows press secretaries to publish posts about media appearances (TV, Radio, Newspapers) and track their statistics.

---

## 🚀 Tech Stack

* Java
* Spring Boot
* PostgreSQL
* REST API

---

## ⚙️ Features

* Create and manage posts
* Attach media links
* Track statistics
* Structured backend architecture
